# Simple Clothing Website

A simple website built using vanilla JavaScript that allows users to view products.

## 🚀 Features

- View all products
- Check price of each product
- View discounts if applicable

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript (Vanilla JS) - JavaScript fundamentals

## 📦 Installation

1. Clone the repository
2. Open index.html in your browser

## 📚 What I Learned

- DOM selection and manipulation
- Show list of products using loop
- Updating UI dynamically

## 📌 Future Improvements

- Add 'Buy Now' feature
- Show 'Thank You' message
